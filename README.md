## 기술 스택

- **Framework:** React 19 + Vite
- **Styling:** CSS (인라인 스타일 + 글로벌 CSS)
- **애니메이션:** IntersectionObserver 기반 스크롤 fade-in, CSS keyframes
- **반응형:** 900px 브레이크포인트 기준 사이드바 ↔ 모바일 햄버거 메뉴 전환

---

## 주요 기능

- **좌측 고정 사이드바** — 네비게이션 링크, 활성 섹션 하이라이트, 수직 스크롤 진행 바
- **우측 섹션 도트 네비** — 현재 위치 실시간 표시, 클릭으로 섹션 이동
- **Scroll Snap** — 섹션 경계에서 자연스러운 스냅 효과
- **모바일 대응** — 햄버거 메뉴 슬라이드 인/아웃, 오버레이
- **Hero 섹션** — 핵심 스탯 카드, 기술 스택 미리보기, 스크롤 유도 인디케이터
